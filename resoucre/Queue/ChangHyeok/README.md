## 큐(Queue)란?
- 기본적인 자료 구조의 한가지로, 먼저 집어 넣은 데이터가 먼저 나오는 FIFO(First In First Out)구조로 저장하는 형식을 말한다.
- 영어 단어 Queue는 표를 사러 일렬로 늘어선 사람들로 이루어진 줄을 말하기도 하며, 먼저 줄을 선 사람이 먼저 나갈수 있는 상황을 연상하면 된다.
- 나중에 집어넣은 데이터가 먼저 나오는 Statck과는 반대되는 개념이다.
